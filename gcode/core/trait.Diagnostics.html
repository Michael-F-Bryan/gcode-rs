Expand description

Callbacks invoked by the parser when it encounters invalid or unexpected input.

The parser does not abort on these conditions; it reports via your implementation and continues. Override the default no-op implementations to collect or log diagnostics. The crate::ast module provides a diagnostics type that implements this trait and accumulates messages.
