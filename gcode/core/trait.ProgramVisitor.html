Expand description

Top-level visitor: one implementation per parse, created by the caller and passed to parse or resume.

The parser calls start_block for each block (line) in the source. Return ControlFlow::Continue with a BlockVisitor to process that line, or ControlFlow::Break to pause (e.g. buffer full); the returned ParserState can be passed to resume to continue later.
